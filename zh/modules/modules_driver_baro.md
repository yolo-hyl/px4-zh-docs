# 模块参考：气压计（驱动程序）

## bmp280

源代码: [drivers/barometer/bmp280](https://github.com/PX4/PX4-Autopilot/tree/main/src/drivers/barometer/bmp280)

### 用法 {#bmp280_usage}

```
bmp280 <命令> [参数...]
 命令:
   start
     [-I]        内部I2C总线
     [-X]        外部I2C总线
     [-s]        内部SPI总线
     [-S]        外部SPI总线
     [-b <val>]  板级专用总线(默认=全部)(外部SPI: 第n个总线(默认=1))
     [-c <val>]  芯片选择引脚(内部SPI)或索引(外部SPI)
     [-m <val>]  SPI模式
     [-f <val>]  总线频率，单位kHz
     [-q]        静默启动(未发现设备时不显示提示)
     [-a <val>]  I2C地址
                 默认值: 118
     [-s]        内部SPI总线
     [-S]        外部SPI总线
     [-b <val>]  板级专用总线(默认=全部)(外部SPI: 第n个总线(默认=1))
     [-c <val>]  芯片选择引脚(内部SPI)或索引(外部SPI)
     [-m <val>]  SPI模式
     [-f <val>]  总线频率，单位kHz
     [-q]        静默启动(未发现设备时不显示提示)

   stop

   status        打印状态信息
```
